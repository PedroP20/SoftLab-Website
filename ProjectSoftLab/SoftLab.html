Sobre

Quais cursos a SoftLab contém?

A SoftLab atualmente trabalha com 3 cursos como opção de entrada no mundo da programação. Eles são: HTML, CSS e JavaScript. O curso de HTML, contém "12" vídeos com duração total de 30min e o curso de CSS com 6 vídeos com duração de 36min. Dentro de cada cursos serão vídeo-aulas sobre conhecimentos básicos de cada assunto, mas bem eficientes para o aluno sair daqui sabendo o necessário.

Por quem a SoftLab foi criada?

A SoftLab foi fundada na Faculdade Metodista de São Paulo, por 7 alunos nos cursos de S.I e ADS. A empresa foi criada como objetivo no 5 semestre de Programa de Extensão Universitária. Cada aluno ficou responsável por uma parte do projeto, que foi desafiador. Os alunos são: Pedro Euphrausino, Victor Massarelli, Caio Silva, Pietro Souza e Paloma Andrade.

O que é HTML?

HTML (Hyper Text Market Language) é a linguagem padrão usada para criar a estrutura e o conteúdo de uma página da web. São usadas tags para definir elementos como cabeçalhos, parágrafos, imagens, links e formulários.

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a aparência visual da página da web, onde o usuário pode mudar a cor dos elementos, selecionar ou mudar a fonte, aumentar o tamanho dos elementos e outras variedades de coisas.

Qual a relação entre HTML e CSS?

A relação entre as duas linguagens é que o HTML define a estrutura e conteúdo de uma página da web, enquanto CSS define a sua aparência visual. Essas duas tecnologias trabalham Juntas para criar uma experiência de usuário consistente e atraente na web, ou seja, uma tag e elemento citado no HTML, é usado o CSS para definir o estilo daquele.

Essas linguagens são fáceis de aprender?

Aprender HTML e CSS pode ser desafiador no começo, mas com prática e dedicação é possível desenvolver habilidades nessas áreas. Existem muitos recursos disponíveis na internet, incluindo tutoriais em vídeo, artigos e cursos online, que podem ajudar a aprender essas habilidades. A chave para aprender é começar com o básico e construir gradualmente sua compreensão e habilidades a partir daí. Como com qualquer coisa, é importante ter paciência e perseverança.
